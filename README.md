
# פרויקט זיהוי פייק ניוז (Fake News Detection)

פרויקט זה עושה שימוש בלמידת מכונה (Machine Learning) כדי לסווג כתבות חדשות כ"אמיתיות" או "מזויפות" (Fake News). המודל מתבסס על ניתוח טקסטואלי של תוכן הכתבה באמצעות אלגוריתם **KNN** וייצוג וקטורי בשיטת **TF-IDF**.

## 📝 סקירה כללית
בעידן המידע, זיהוי אוטומטי של פייק ניוז הוא כלי קריטי למניעת הפצת דיסאינפורמציה. הפרויקט מדגים תהליך מלא של Data Science:
1. עיבוד מקדים של טקסט (Preprocessing).
2. הנדסת מאפיינים (Feature Engineering).
3. בחירת מודל ואימון.
4. אופטימיזציה של היפר-פרמטרים (Hyperparameter Tuning).
5. ניתוח שקיפות המודל (Explainability).

## 📊 נתונים (Dataset)
הפרויקט משתמש בקבצי `True.csv` ו-`Fake.csv`.
* **אמיתי (True):** כתבות חדשות מאומתות (לרוב ממקורות כמו רויטרס).
* **מזויף (Fake):** כתבות ממקורות לא אמינים או תוכן שהוגדר כפייק.

## 🛠️ טכנולוגיות וכלים
* **Python**
* **Scikit-learn:** אימון המודל, TF-IDF וניהול ניסויים.
* **Pandas & NumPy:** מניפולציה של נתונים וחישובים.
* **Jupyter Notebook / Google Colab:** סביבת הפיתוח.

## 🚀 שלבי העבודה
1. **ניקוי נתונים:** הסרת מילות קישור (Stop Words) באנגלית וסינון תווים מיותרים.
2. **ייצוג טקסט:** שימוש ב-`TfidfVectorizer` להמרת טקסט חופשי למטריצה מספרית המבטאת את חשיבות המילים.
3. **בחירת מודל:** שימוש באלגוריתם **K-Nearest Neighbors (KNN)**.
4. **ניהול ניסויים:** * ביצוע **Grid Search** למציאת הפרמטרים האופטימליים.
   * שימוש ב-**5-Fold Cross Validation** להבטחת עמידות המודל.
   * בדיקת פרמטרים כמו מספר שכנים (K) ומשקלים (Uniform vs Distance).
5. **שקיפות (Explainability):** הצגת השכנים הקרובים ביותר לכתבה נתונה וזיהוי מילות המפתח שהשפיעו על החלטת המודל.

## 📈 תוצאות
המודל הסופי הגיע לביצועים מרשימים:
* **F1-Score:** כ-0.80 (על סט הבדיקה).
* המודל הראה שיפור משמעותי לאחר ביצוע אופטימיזציה של הפרמטרים והגדלת כמות המאפיינים.
